Project Battleship IV
==========
basic programming in python

This is an extended version of the Codecademy exercise Project Battleship

Upgrades include:

1, Making the game re-playeble
2, 2 Player mode option
3, AI mode option
4, Checking the correctness of users input, including empty input
5, Two ships
6, Larger ships (4 tiles and 2 tiles)
7, Numbered grid of the ocean
8, Difficulty levels (changing the board size and the number of tries)
